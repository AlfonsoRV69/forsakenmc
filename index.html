ForsakenMC

ForsakenMC is a mod for Minecraft Java Edition for NeoForge 1.21.1 that brings content from the popular Roblox game Forsaken into Minecraft! Experience new adventures and challenges inspired by Forsaken’s world.

Main Features

  • Adds weapons, mobs, items, and mechanics from Roblox's Forsaken
  • Adds a new structure from Forsaken
  • Fight custom and challenging Forsaken mobs

Installation

  1. Install NeoForge 1.21.1 for Minecraft.
  2. Download the .jar file of ForsakenMC from the releases page.
  3. Place the file in your Minecraft mods folder.
  4. Start the game and enjoy!
